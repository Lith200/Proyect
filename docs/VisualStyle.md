> Está en cómo todos trabajan juntos."

# Identidad Visual

Project Moon combina tres conceptos:

🌌 Espacio profundo

💜 Romance

✨ Magia

No debe parecer una página web.

Debe sentirse como una experiencia interactiva.

# Filosofía del Color

Los colores no deben competir.

Cada color tiene un propósito.

Azules → calma

Morados → misterio

Rosas → amor

Dorados → momentos especiales

Blancos → magia

---

# Iluminación

Toda luz debe parecer natural.

Nunca usar sombras negras.

Las sombras siempre tendrán un ligero tono azul o morado.

Los brillos serán suaves.

Difusos.

Nunca agresivos.

---

# Materiales

Todo debe sentirse construido con:

Cristal

Luz

Nebulosa

Polvo espacial

Nunca plástico.

Nunca metal pesado.

---

# Cristal Espacial

Las tarjetas deben parecer:

Vidrio de una nave espacial.

Características:

Blur suave.

Borde blanco muy fino.

Reflejos sutiles.

Transparencia elegante.

---

# Planetas

Todos los planetas deben tener:

Iluminación lateral.

Sombra.

Atmósfera.

Reflejo.

Volumen.

Nunca colores planos.

---

# Nebulosas

Siempre enormes.

Muy transparentes.

Nunca perfectamente redondas.

Con formas orgánicas.

---

# Estrellas

Tres tamaños.

Nunca alineadas.

Nunca distribuidas en una cuadrícula.

Siempre aleatorias.

---

# Cometas

Muy elegantes.

Nunca aparecen constantemente.

Siempre dejan una cola luminosa.

---

# Partículas

Muy pequeñas.

Muy transparentes.

Movimiento lento.

No deben distraer.

Solo dar vida.

---

# Tipografía

Debe sentirse:

Limpia.

Moderna.

Elegante.

Respirable.

Con espacio.

Nunca comprimida.

---

# Animaciones

Todas las animaciones siguen estas reglas:

Nunca bruscas.

Nunca rápidas.

Nunca robóticas.

Todo debe acelerar y desacelerar naturalmente.

---

# Movimiento

Todo objeto vivo tiene un movimiento.

Muy pequeño.

Muy lento.

Muy elegante.

---

# Profundidad

Todo debe sentirse en capas.

Nunca plano.

Cada elemento pertenece a un plano diferente.

---

# Principio del Vacío

El espacio vacío también comunica.

No llenar toda la pantalla.

Respirar también es diseño.

---

# Cinemática

Cada pantalla debe sentirse como una escena de película.

No como una página.

---

# Regla de Oro

Antes de agregar un efecto debemos responder:

¿Hace que el usuario sienta algo?

Si la respuesta es NO...

Ese efecto no entra al proyecto.

---

# Objetivo Final

Queremos que alguien abra Project Moon y piense:

"No parece una página web...

Parece un pequeño universo hecho especialmente para mí."

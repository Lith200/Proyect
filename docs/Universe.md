# Universe Design Document (UDD)

> "No estamos creando una página web.
> Estamos creando un pequeño universo donde cada estrella,
> cada planeta y cada movimiento cuentan una historia."

# Versión

Legendary Edition

---

# Filosofía

El universo de Project Moon representa el viaje emocional de una declaración de amor.

No es un espacio vacío.

Es un lugar vivo.

Cada elemento tiene un propósito.

Cada color transmite una emoción.

Cada movimiento debe sentirse natural.

Nunca exagerado.

Nunca artificial.

Todo debe transmitir calma, esperanza, magia y amor.

---

El usuario debe sentir:

🌌 inmensidad

# Principios del Universo

## 1.

El universo siempre está vivo.

Aunque el usuario no toque nada,
el universo continúa respirando.

---

## 2.

Nunca existe un momento completamente estático.

Siempre ocurre algo.

Una estrella parpadea.

Una partícula flota.

Una nebulosa se mueve.

Un planeta gira.

---

## 3.

Nada debe moverse rápido.

El espacio transmite calma.

Todo movimiento será lento y elegante.

---

## 4.

La iluminación debe parecer real.

Nunca usar colores completamente planos.

Todo planeta debe tener:

• luz

• sombra

• atmósfera

• profundidad

• reflejos

---

## 5.

Cada pantalla representa una etapa emocional.

El universo también cambia junto con la historia.

---

# Arquitectura del Universo

El universo está compuesto por capas.

Cada capa tiene una función.

Nunca deben mezclarse responsabilidades.

---

# CAPA 01

Deep Space

Representa:

El infinito.

Movimiento:

Ninguno.

Color principal:

Negro espacial.

Azul profundo.

Morado oscuro.

Nunca completamente negro.

---

# CAPA 02

Nebulosas

Representan:

Emoción.

Características:

• enormes

• suaves

• difusas

• transparentes

• movimiento extremadamente lento

Nunca deben llamar demasiado la atención.

Son el fondo del universo.

---

# CAPA 03

Campo Estelar

Tres niveles.

Nivel 1

Estrellas lejanas.

Muy pequeñas.

Miles.

Brillo muy bajo.

---

Nivel 2

Estrellas medias.

Más visibles.

Parpadeo ocasional.

---

Nivel 3

Estrellas cercanas.

Grandes.

Muy pocas.

Brillo intenso.

---

# CAPA 04

Polvo Espacial

Pequeñas partículas flotando.

Movimiento aleatorio.

Muy lento.

Casi invisible.

Debe sentirse como si el universo estuviera lleno de materia.

---

# CAPA 05

Cometas

Eventos especiales.

Nunca aparecen constantemente.

Características:

• dirección aleatoria

• velocidad aleatoria

• brillo variable

• longitud variable

• frecuencia impredecible

Los cometas deben sorprender.

---

# CAPA 06

Planetas

Los planetas representan emociones.

Nunca son decoración.

Cada uno tiene personalidad.

---

Planeta Dorado

Representa:

Esperanza.

---

Planeta Rosa

Representa:

Amor.

---

Planeta Azul

Representa:

Confianza.

---

Planeta Morado

Representa:

Destino.

---

Todos deben tener:

• volumen

---

# CAPA 07

Lunas

Cada luna debe:

orbitar

proyectar sombra

tener iluminación

acompañar a su planeta

Nunca existir por sí sola.

---

# CAPA 08

Glow

El brillo nunca debe cubrir la pantalla.

Debe ser elegante.

Muy controlado.

Pequeños destellos.

Nunca manchas gigantes.

---

# CAPA 09

Magia

Aquí vive la esencia romántica.

Incluye:

✨ auroras

💖 constelaciones

🌠 estrellas fugaces

🌌 nebulosas rosas

✨ destellos dorados

Esta capa aparece poco a poco conforme avanza la historia.

---

# Universo Dinámico

El universo nunca se repite exactamente.

Durante la experiencia pueden ocurrir eventos aleatorios.

Ejemplos:

• pasa un cometa

• aparece una estrella fugaz

• aumenta ligeramente el brillo de una nebulosa

• una constelación comienza a iluminarse

• una lluvia de meteoros aparece durante unos segundos

Todo debe sentirse natural.

Nunca repetitivo.

---

# Respiración del Universo

Todo el universo realiza una respiración muy lenta.

Inspirar.

Pausa.

Expirar.

Pausa.

Este movimiento es casi imperceptible.

Pero hace que el universo se sienta vivo.

# ❤️ PROJECT MOON
# Emotion Engine (EE)

> "Las personas olvidan lo que vieron.
> Olvidan lo que escucharon.
> Pero nunca olvidan cómo las hiciste sentir."

---

# Misión

Project Moon no busca impresionar con efectos.

Busca crear un viaje emocional.

Cada pantalla debe provocar una emoción específica.

El universo debe reaccionar junto con esas emociones.

---

# Filosofía

Las emociones son el motor principal del proyecto.

Los efectos visuales existen únicamente para reforzar esas emociones.

Nunca al revés.

---

# Curva Emocional

La experiencia completa debe sentirse como una película.

No debe iniciar en el punto más alto.

Debe crecer poco a poco.

Cada pantalla aumenta ligeramente la intensidad emocional.

# Regla Suprema

# Objetivo Final

No queremos que la persona diga:

"Qué bonita página."

"Nunca había visto una declaración de amor así."

Ese es el verdadero objetivo de Project Moon.
